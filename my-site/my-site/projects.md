          <div class="details-section">
            <h4>Logic Gates</h4>
            <ul class="logic-gates-list">
              <li><strong>AND Gate:</strong> Outputs 1 only when both inputs are 1</li>
              <li><strong>OR Gate:</strong> Outputs 1 when at least one input is 1</li>
              <li><strong>NOT Gate:</strong> Inverts the input (1 becomes 0, 0 becomes 1)</li>
              <li><strong>NAND Gate:</strong> NOT AND - outputs 0 only when both inputs are 1</li>
              <li><strong>NOR Gate:</strong> NOT OR - outputs 1 only when both inputs are 0</li>
            </ul>
          </div>
